Architecture (short)

EPInformer-abc — attention over promoter–enhancer sequence embeddings with ABC-driven weighting of enhancer features (distance / activity / contact) before expression readout.

EPInformer-v2 — alternative fusion: per-channel feature MLPs and multiplicative gating with sequence embeddings, then transformer layers; training uses the same --model_type EPInformer-v2 flag in train_EPInformer_abc.py.
